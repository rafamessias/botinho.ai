Step 1: Get Your API Credentials

To use Opineeo surveys, you'll need:

  • API Token: Your authentication token
  • Survey ID: The ID of the survey you want to display

Step 2: Include the Script

Add the Opineeo script to your HTML:

<script src="https://cdn.opineeo.com/opineeo-0.0.1.min.js"></script>

Step 3: Add the Survey

Use the web component tag anywhere in your HTML:

<opineeo-survey
  survey-id="your-survey-id"
  token="your-api-token">
</opineeo-survey>
