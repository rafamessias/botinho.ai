üöÄ Quick Start

Get up and running in under 2 minutes

1. Include the Script

Add the Opineeo widget script to your HTML page.

<script src="opineeo-sv-w.js"></script>

2. Use Web Component (Tag-based)

Simply add the custom element to your HTML.

<opineeo-survey survey-id="my-survey" token="your-api-token" auto-close="3000"> </opineeo-survey>

3. Programmatic Usage

Create the component dynamically with JavaScript.

// Create element programmatically const surveyEl = document.createElement('opineeo-survey'); surveyEl.setAttribute('survey-id', 'my-survey'); surveyEl.setAttribute('token', 'your-api-token'); // Set survey data surveyEl.surveyData = { id: 'my-survey', name: 'Customer Feedback', questions: [/* your questions */] }; // Add event listeners surveyEl.addEventListener('complete', (e) => { console.log('Survey completed!', e.detail); }); // Append to page document.body.appendChild(surveyEl);

4. Done! üéâ

Your survey widget is now live and ready to collect responses from your users.

Access and process survey responses:

üéõÔ∏è Multiple Instances

Run multiple surveys on the same page:

Please try again.'); // Log to analytics analytics.track('survey_error', { error: error.message }); } });
