🚀 Quick Start

Get up and running in under 2 minutes

1. Include the Script

Add the Opineeo widget script to your HTML page.

<script src="opineeo-sv-w.js"></script>

2. Create Container

Add a container element where the survey will be displayed.

<div id="my-survey"></div>

3. Initialize Widget

Create and configure your survey widget with sample data.

const widget = window.initSurveyWidget({ surveyData: { id: 'my-survey', name: 'Customer Feedback', questions: [/* your questions */] }, onComplete: (responses) => { console.log('Survey completed!', responses); } }); widget.mount('my-survey');

4. Done! 🎉

Your survey widget is now live and ready to collect responses from your users.

Complete Implementation Example

Here's a full example showing how to implement the survey widget with custom styling and event handling:

<!-- Include the widget script --> <script src="opineeo-sv-w.js"></script> <!-- Create survey container --> <div id="customer-feedback"></div> <script> // Initialize the survey widget const surveyWidget = window.initSurveyWidget({ surveyData: { id: 'customer-feedback-2024', name: 'Customer Satisfaction Survey', description: 'Help us improve our service', status: 'published', questions: [ { id: 'satisfaction', title: 'How satisfied are you with our service?', description: 'Please rate your overall experience', format: 'STAR_RATING', required: true, order: 1, options: [] }, { id: 'contact_method', title: 'What is your preferred contact method?', format: 'SINGLE_CHOICE', required: true, order: 2, options: [ { id: 'email', text: 'Email', order: 1, isOther: false }, { id: 'phone', text: 'Phone', order: 2, isOther: false }, { id: 'chat', text: 'Live Chat', order: 3, isOther: false }, { id: 'other', text: 'Other', order: 4, isOther: true } ] }, { id: 'features', title: 'What features would you like to see?', format: 'MULTIPLE_CHOICE', required: false, order: 3, options: [ { id: 'mobile', text: 'Mobile App', order: 1, isOther: false }, { id: 'ui', text: 'Better UI/UX', order: 2, isOther: false }, { id: 'integrations', text: 'More Integrations', order: 3, isOther: false } ] }, { id: 'comments', title: 'Any additional comments?', format: 'LONG_TEXT', required: false, order: 4, options: [] } ] }, customCSS: ` .sv { background: #ffffff; color: #1f2937; margin: 16px 0px; padding: 24px; border: 1px solid #e5e7eb; border-radius: 12px; font-family: Inter, sans-serif; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.05); } .qt { font-size: 20px; font-weight: 600; color: #1f2937; } .qd { font-size: 16px; color: #6b7280; } .btnp { background: #3b82f6; color: #ffffff; border-radius: 8px; padding: 12px 24px; font-weight: 500; transition: all 0.3s ease; } .btnp:hover { background: #2563eb; transform: translateY(-1px); } `, autoClose: 3000, onComplete: (responses) => { console.log('Survey completed!', responses); // Send responses to your server fetch('/api/survey-responses', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(responses) }); }, onError: (error) => { console.error('Survey error:', error); } }); // Mount the survey to the container surveyWidget.mount('customer-feedback'); </script>

❓ Question Types

All supported question formats with examples

⭐ Star Rating

Perfect for satisfaction surveys and ratings.

{ "format": "STAR_RATING", "title": "Rate our service", "required": true, "options": [] }

✅ Yes/No Questions

Simple binary choice questions with custom labels.

{ "format": "YES_NO", "title": "Would you recommend us?", "yesLabel": "Yes, definitely", "noLabel": "Not yet", "required": true, "options": [] }

🔘 Single Choice

Radio button selection with optional "Other" field.

{ "format": "SINGLE_CHOICE", "title": "How did you hear about us?", "options": [ { "id": "google", "text": "Google Search" }, { "id": "social", "text": "Social Media" }, { "id": "other", "text": "Other", "isOther": true } ] }

☑️ Multiple Choice

Checkbox selection allowing multiple answers.

{ "format": "MULTIPLE_CHOICE", "title": "What features interest you?", "options": [ { "id": "feature1", "text": "Mobile App" }, { "id": "feature2", "text": "Analytics" }, { "id": "feature3", "text": "Integrations" } ] }

📝 Long Text

Free-form text input for detailed feedback.

{ "format": "LONG_TEXT", "title": "Additional comments", "description": "Tell us more about your experience", "required": false, "options": [] }

📄 Statement

Display informational content or consent text.

{ "format": "STATEMENT", "title": "Privacy Notice", "description": "Your responses are confidential and will be used to improve our service.", "options": [] }

🎯 CSS Scoping

Each survey gets its own CSS scope to prevent conflicts:

/* Your CSS is automatically scoped to the survey */ .sv-scope-my-survey .sv { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 20px; } .sv-scope-my-survey .qt { color: white; font-size: 24px; } .sv-scope-my-survey .btnp { background: #ff6b6b; border-radius: 25px; }

🎨 Custom Styling Guide

Key CSS classes you can customize:

/* Main container */ .sv { /* Survey container */ } /* Typography */ .qt { /* Question title */ } .qd { /* Question description */ } /* Buttons */ .btnp { /* Primary button (Next/Submit) */ } .btno { /* Secondary button (Previous) */ } /* Input elements */ .rad { /* Radio button container */ } .chk { /* Checkbox container */ } .txt { /* Text input */ } .ta { /* Textarea */ } /* Star rating */ .stars { /* Star container */ } .star-btn { /* Individual star button */ }

📱 Responsive Design

Built-in responsive breakpoints for all devices:

/* Mobile-first approach */ @media (max-width: 400px) { .sv { max-width: 100%; margin: 8px; padding: 16px; } .qt { font-size: 16px; } .star { font-size: 20px; } }

⚡ Advanced Features

Powerful features for complex use cases

🔄 Auto-close Timer

Automatically close the survey after completion:

const widget = window.initSurveyWidget({ surveyData: surveyData, autoClose: 3000, // Close after 3 seconds onComplete: (responses) => { console.log('Survey completed!', responses); } });

📊 Response Handling

Access and process survey responses:

onComplete: (responses) => { responses.forEach(response => { console.log(`Question: ${response.questionTitle}`); console.log(`Value: ${response.textValue || response.numberValue || response.booleanValue}`); console.log(`Option ID: ${response.optionId}`); }); // Send to your server fetch('/api/survey-responses', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(responses) }); }

🎛️ Multiple Instances

Run multiple surveys on the same page:

// First survey const survey1 = window.initSurveyWidget({ surveyData: { ...data1, id: 'survey-1' }, customCSS: '/* styles for survey 1 */' }); survey1.mount('container-1'); // Second survey with different styling const survey2 = window.initSurveyWidget({ surveyData: { ...data2, id: 'survey-2' }, customCSS: '/* different styles for survey 2 */' }); survey2.mount('container-2');

🛡️ Error Handling

Handle errors gracefully:

const widget = window.initSurveyWidget({ surveyData: surveyData, onError: (error) => { console.error('Survey error:', error); // Show user-friendly message alert('Sorry, there was an error loading the survey. Please try again.'); // Log to analytics analytics.track('survey_error', { error: error.message }); } });
